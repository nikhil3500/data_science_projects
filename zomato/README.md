Project Title: Zomato Restaurant Data Analysis and Dashboard

This project analyzes Zomato's restaurant data using Tableau, combining location and rating datasets to uncover actionable insights into restaurant performance, customer preferences, and market trends. Key deliverables include interactive dashboards and visualizations to support data-driven decision-making.

Key Features:
Data Joining: Combined "Location" and "Rating" tables using restaurant IDs to create a unified dataset.
Visualizations:
City-wise restaurant distribution (table).
Heat map for average city ratings.
Average cost analysis by city.
Franchise distribution and top 10 restaurants (packed bubble chart).
Best 5 restaurants by ratings with a minimum threshold of 10 votes (bar chart).
KPIs: Total restaurants, average cost, average rating, and total votes.
Dashboard & Story: Developed an interactive dashboard with filters and actions, formatted per Zomato's theme, and created a cohesive story to present insights.
